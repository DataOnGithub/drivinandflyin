tomtom.controls.OverviewMap Class

Extends OpenLayers.Control.OverviewMap
Defined in: src/main/webapp/js/tomtom/controls/OverviewMap.js:14

Creates an overview map which allows the user to easily navigate the map. Please note that this constructor shouldn't be called directly. The overviewMap property of the main map constructor's options should be used to display and customize the overview map like so:

var map = new tomtom.Map({
    domNode: "map",
    overviewMap: true
});

You can also pass custom options:

var map = new tomtom.Map({
    domNode: "map",
    overviewMap: {
        size: new OpenLayers.Size(300, 300)
    }
});
