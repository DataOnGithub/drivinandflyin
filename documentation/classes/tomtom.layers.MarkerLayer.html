tomtom.layers.MarkerLayer Class

Extends OpenLayers.Layer.Markers
Defined in: src/main/webapp/js/tomtom/layers/MarkerLayer.js:12

Creates a marker layer which can be used to add markers to the map. A map can have multiple marker layers. The map creates a "default" marker layer which is used when the addMarker method is called.

A marker layer can be added just like any other layer:

var layer = new tomtom.layers.MarkerLayer("MyMarkerLayer");
map.addLayer(layer);
