setPopup

(
  • popupClass
  • options
  • [event=click]
)

Inherited from tomtom.Marker: src/main/webapp/js/tomtom/Marker.js:152

Associates a popup with the marker. This will allow a popup to be shown when the specified event occurs.

For example:

var marker = new tomtom.Marker(-71.53686, 43.20846);
marker.setPopup(tomtom.Popup, { contentHTML: "My Popup!" }, "click");

This will create a standard popup that is displayed when the user clicks on the marker.

Parameters:

  • popupClass Constructor

    The popup constructor to use to create the popup. Please note that the popup class must inherit from .

  • options Object

    The options argument to be supplied to the popup's constructor.

  • [event=click] String optional

    The event which will cause the popup to be displayed.
