- (NSString *) replaceExtraParameters: (NSString*)  request
withQuery: (NSString*)  query 

Template method to replace any extra parameters in the URL.

By default replaces ${query} with the value in query. If the query is nil it does not perform any replacement. An empty string can be used as input.

Parameters:
request String in which the replacement will take place. Already has the SDK's api key.
query The query value passed by parameter to the init method.
Returns:
The concrete URL, fully parameterized.
