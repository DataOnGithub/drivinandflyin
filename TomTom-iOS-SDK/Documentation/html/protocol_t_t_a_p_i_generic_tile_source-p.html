- (id) initWithBaseUrl: (NSString *)  baseUrl

Inits a source tile class with URL and params.

This method receives the base URL from which the SDK should be getting the tile images along with parameters that should be general to all calls. Specific parameters for each call should be passed with the tileUrl method.

Parameters:
baseUrl The base URL from where the SDK gets the tile images.
params Generic parameters that the class will use to initialize the object. Should be general for all calls to the base URL.
Returns:
The generic tile source that is used throughout the SDK.
