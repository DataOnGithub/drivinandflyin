CGPoint metersToTileXy ( CGPoint  meters,
int  zoom 
)

Converts meters into a corresponding (x,y) tile for the given zoom level.

Since there is no direct correspondency between a tile's (x,y) coordinates and geo meters, the tile returned will be the tile (for the given zoom level) that contains the geo point location. The upper left corner of the tile may not correspond to the geo location passed as parameter.

Parameters:
meters Geo meters of the intended location.
zoom Zoom level for which the tile (x,y) should be calculated.
Returns:
The CGPoint corresponding to the (x,y) coordinates of the tile.
