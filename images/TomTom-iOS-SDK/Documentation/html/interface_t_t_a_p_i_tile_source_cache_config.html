- (id) initwithCacheKey: (NSString *)  cacheK
andCapacities: (NSMutableArray*)  caps
andTypes: (NSMutableArray*)  types 

Inits and returns a cache source configuration object.

The elements of the caps array will have a direct corespondency to the elements of the types array.

Parameters:
cacheK Cached key to be used to uniquely identify the caches that are being initialized.
caps Capacities to be used by the caches that will be created with the tile source.
types Types of caches to be used by the tile sources. The only possible values are 'memory-cache' and 'db-cache'.
Returns:
Returns the new cache configuration object.
