Detailed Description

Defines the optional parameters for the geocode requests.

Member Function Documentation

- (NSString *) stringValue  

Converts the parameters to a query string that can be added to the end of the URL that accesses the server.

Accounts for non initialized parameters.

Returns:
An NSString with the parameters in the format: ";fullCopyright;map=mapType"

Property Documentation

- (TTBBox *) bbox [read, write, retain]

A bounding box to which the request should be restricted.

Specified as min_lat,min_lon,max_lat,max_lon in the coordinate system of the projection used (see projection below).

- (TTCoordinates *) biasPoint [read, write, retain]

The point from where the results should be ordered.

This defaults to the center of bbox.

- (NSString *) geoHash [read, write, retain]

The query geohash.

- (NSString *) language [read, write, retain]

The language of the response.

Default is en (English).

- (TTLocationOrder) order [read, write, assign]

The sort order of the results.

Possible values are SCORE (default) and PROXIMITY.

- (NSInteger) poiCategories [read, write, assign]

The categories of the POIs the request should look for.

- (TTProjection) projection [read, write, assign]

The map projection used to specify coordinates returned in the response.

Possible values are EPSG7059 (default) and EPSG900913.
