Property Documentation

- (BOOL) avoidTolls [read, write, assign]

Boolean flag to indicate whether or not to avoid toll roads when planning the route.

Default value is false.

- (BOOL) avoidTraffic [read, write, assign]

Boolean flag to indicate if traffic information is used to plan the route (only relevant for routeType Quickest and AvoidMotorway).

Default value is false.

- (int) dataPathZoomLevel [read, write, assign]

zoom level for the data path.

- (TTRouteDayType) day [read, write, assign]

The day of departure.

Possible values are today, tomorrow, monday, tuesday, wednesday, thursday, friday, saturday, and sunday. Default value is today.

- (BOOL) includeDataPath [read, write, assign]

Boolean flag to indicate if the complete vector data path of the route must be returned.

- (BOOL) includeInstructions [read, write, assign]

Boolean flag to indicate if turn-by-turn instructions should be returned in the response.

Default value is true.

- (BOOL) includeTraffic [read, write, assign]

Boolean flag to indicate if traffic information is used to describe delays in the route (not relevant for routeType Walk and Bicycle).

Default value is false.

- (TTRouteIQType) iqRoutes [read, write, assign]

The type of IQ Routes to use when planning the route.

Possible values are none, Charleston – take only day into account), and Seattle – take day & time into account. Default value is none.

- (NSString *) language [read, write, retain]

The language to use for the route instructions.

Default is en (English).

- (TTProjection) projection [read, write, assign]

The map projection used to specify coordinates, both for the startPoint:endPoint and data returned in the response.

Possible values are EPSG7059 (default) and EPSG900913.

- (NSString *) time [read, write, retain]

The time of departure.

Possible values are anytime, now, and 0...1439 (time expressed as the number of minutes since midnight). Default value is now.

- (NSString *) trafficModelId [read, write, retain]

String representation of the traffic model ID to use in the route calculation.

If this parameter is omitted, the latest ID is used.
